希望 QTM 调研时重点覆盖：
1. Agent-to-agent / machine-to-machine payment 的主要候选形态：稳定币、链上微支付、账户抽象/智能钱包、传统银行卡/API扣费、平台积分/预付余额、银行开放接口、Lightning、CBDC、专用协议等。

3. 目前已经有人在尝试的项目/协议/公司/论文/标准：请尽量给出名称、做法、成熟度和可信来源线索。
4. 输出要形成“当前结论 + 证据地图 + 未来3种可能路径 + 对AI教育/内容/服务agent商业化的启发”。
