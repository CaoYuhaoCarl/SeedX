# 学习问题

qtm生成宁波区域舆情监控系统

## 显式目标

请围绕“宁波区域舆情监控系统”生成一套从问题理解到可执行建设路径的学习/方案化产物，帮助学习者掌握该系统的需求框架、数据来源、架构设计、监控指标、预警流程、运营机制与落地路线。

## 安全与合规约束

- 仅讨论公开信息、授权数据与合规数据源；不得设计或鼓励绕过平台规则、侵犯隐私、非授权采集、个人跟踪、画像歧视或人肉搜索。
- 系统定位为区域公共信息态势感知、风险预警与应急协同辅助工具；不得用于压制合法表达、定向骚扰或扩大监控。
- 必须包含数据最小化、脱敏/匿名化、访问控制、审计日志、人工复核、误报纠偏、公众权益保护与合规评估机制。
- 输出应明确区分 MVP、增强版与长期演进，不承诺自动判断事实真伪或替代人工决策。
