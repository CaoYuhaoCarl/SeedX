# 学习问题

我想快速掌握 AI Agent memory 的基本框架，并能判断它适合解决哪些真实问题。

## 背景

我已经知道大模型可以在对话中使用上下文，但不清楚 Agent memory 和普通上下文、用户画像、RAG、日志记录之间有什么区别。

## 目标

1. 理解 Agent memory 的核心概念和边界。
2. 理解 memory 写入、检索、更新、遗忘的基本机制。
3. 能判断一个 Agent 项目是否真的需要 memory。
4. 能设计一个最小可行的 memory schema。

## 时间约束

希望在 2 小时内建立可用框架，不追求生产级实现细节。

## 输出偏好

希望输出包含：

- 核心概念地图
- 分阶段学习路径
- 具体练习
- 掌握检查
- 通用应用与迁移计划
